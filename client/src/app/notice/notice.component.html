DPS sim for SCH based on the original Google sheets sim is available: change stat values, party comp, and rotation info to see the effect on DPS.
Changes you make will NOT persist if you close the browser tab.
Future updates will allow gearsets to be saved/loaded, custom rotations, support for additional jobs, and more.
If you notice any issues or have feedback, please use the button in the lower left to visit us on GitHub, check to see if your issue is already tracked, and file one if it's not. Thanks!
